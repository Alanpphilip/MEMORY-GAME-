# Memory Game - 2 Players

This is a browser-based 2-player Memory Card Game built using HTML, CSS, and JavaScript.

## Features
- Player 1 starts first
- Players take turns
- Correct match gives another turn
- Wrong match switches turn
- Score tracking
- Winner announcement
- Restart button

## How to play
Flip cards and match pairs. The player with the highest score wins.
